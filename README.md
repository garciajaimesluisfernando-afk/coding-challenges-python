# Coding Challenges & Self-Assessments

## Overview
This repository contains a collection of coding challenges and technical exams that I design and implement for myself. The goal of this project is to test my problem-solving abilities, logic, and mastery of data structures under mock exam conditions.

## Contents
* **examen listas enlazadas.py**: A custom self-assessment focusing on the implementation, manipulation, and traversal of linked list nodes using Python.

## Technical Skills Demonstrated
* Object-Oriented Programming (OOP) in Python.
* Pointer manipulation and dynamic memory logic.
* Code design without external frameworks.
